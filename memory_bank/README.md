# Memory Bank

The Memory Bank is the agent's persistent project context. Because an AI assistant's context resets
between sessions, these files let it rebuild understanding quickly. **Read these at the start of a
session; update them when decisions change.**

| File | Purpose |
| --- | --- |
| [`projectBrief.md`](projectBrief.md) | Scope, goals, and success criteria |
| [`productContext.md`](productContext.md) | Why the project exists; the users and problems |
| [`systemPatterns.md`](systemPatterns.md) | Architecture and key implementation decisions |
| [`techContext.md`](techContext.md) | Stack, runtimes, and technical constraints |
| [`activeContext.md`](activeContext.md) | Current focus and open questions |
| [`progress.md`](progress.md) | What works and what's left |
| [`LEARNING_LOG.md`](LEARNING_LOG.md) | Decisions, corrections, and conventions (ADRs) |

> These are starter templates — fill in the `_TODO_` placeholders for your project.
